HTML5 - Hyper Text Markup Language

1.What is HTML?

HTML stands for HyperText Markup Language. It is the standard markup language used to create and design the structure of web pages.

2.What is meta tag?

The meta tag is used to provide metadata about the HTML document, such as character encoding, author, description, and keywords. It is typically placed in the head section of the page.

3.What is semantic tag?

Semantic elements are HTML tags that carry meaning about the structure of the page and its content. Examples include header, nav, main, section, article, footer, etc.

4.What is a Html Web Worker?

A web worker is a JavaScript running in the background, without affecting the performance of the page.

5.What is the role of the iframe tag?

The iframe tag is used to embed another document within the current HTML document. It is commonly used to embed videos, maps, or external web content.

6.What is HTML5, and what are its new features?

HTML5 is the latest version of the HyperText Markup Language used for structuring and presenting content on the web. Its new features include support for audio and video elements, canvas for graphics, new form input types, local storage, geolocation, and semantic elements.

7.What is the difference between HTML elements and HTML tags?

HTML tags are used to define the beginning and end of HTML elements. An HTML element consists of a starting tag, content, and an ending tag (if applicable).

8.Explain the difference between SVG and canvas for graphics in HTML5.

SVG (Scalable Vector Graphics) is an XML-based format used to define graphics and is resolution-independent, while canvas allows dynamic rendering of graphics using JavaScript and is resolution-dependent.

9.What is the purpose of the viewport meta tag in HTML5?

The viewport meta tag controls the layout and scaling of a web page on different devices, especially mobile devices, ensuring responsive design.

10.Explain the difference between Web Storage and Cookies in HTML5.

Web Storage (localStorage and sessionStorage) provides a more efficient and secure way to store data on the user's browser than traditional cookies. Web Storage data is not sent with every HTTP request, reducing overhead.

CSS3 - Cascading Style Sheets

1.What is CSS?

CSS stands for Cascading Style Sheets. It is a style sheet language used to control the presentation and layout of HTML documents.

2.How do you include CSS in an HTML document? Explain different methods.

There are three main ways to include CSS in an HTML document:

  • Inline CSS : You can apply styles directly to an HTML element using the "style" attribute.
  • Internal CSS : You can define CSS styles within the "style" element in the head section of the HTML document.
  • External CSS : You can create a separate CSS file with all the styles and link it to your HTML document using the "link" element.

3.What are the different CSS selectors?

CSS selectors are patterns used to select and style HTML elements.

  • Type Selector(Element Selectors)
  • Class Selector
  • ID Selector
  • Descendant Selector
  • Attribute Selector

4.Explain the CSS Box Model and its components.

The CSS Box Model is a fundamental concept that describes how the space around an HTML element is calculated. It consists of four components:

  1. Content : The actual content area of the element, where text, images, or other media are displayed.
  2. Padding : The space between the content and the element's border.
  3. Border : A border surrounding the padding and content.
  4. Margin : The space between the element's border and other elements on the page.

5.Describe the difference between "display: block," "display: inline," and "display: inline-block."

  • display: block : This property take up the full width available and start on a new line. They ignore the width and height properties of their content.
  • display: inline : This property only take up content width as necessary and do not start on a new line. They respect the width and height properties of their content.Inline elements flow within the content and can appear next to each other on the same line.
  • display: inline-block : This property are similar to inline elements but can have specified width and height properties. They do not start on a new line.elements that can flow inline but still have a specific size and spacing.

6.What is the importance of the "z-index" property?

The "z-index" property is used to control the stacking order of positioned elements on a webpage.It only works on positioned elements (position: absolute, position: relative, position: fixed).

7.What is the purpose of media queries in CSS, and how are they used for responsive web design?

They are essential for creating responsive web designs that adapt to different screen sizes, ensuring a better user experience on various devices like desktops, tablets, and smartphones.

8.Explain the concept of CSS specificity and how it determines which styles are applied to an element.

  • Inline styles (style attribute): Highest specificity
  • ID selectors
  • Class selectors, attribute selectors, and pseudo-classes
  • Type selectors and pseudo-elements: Lowest specificity

9.Position property in CSS ?

  • position: static : This is the default positioning value for all elements.
  • position: relative : Positioned relative to its normal position, and other elements are not affected.
  • position: absolute : Positioned relative to its nearest positioned ancestor or the document body. It is removed from the normal document flow.
  • position: fixed : Positioned relative to the viewport. It is removed from the normal document flow and remains fixed even when scrolling.
  • position: relative : The element is treated as position: relative until it reaches a specified offset position (e.g., top, bottom, left, or right). After that, it behaves like position: fixed; and remains fixed to the specified position.

10.Difference between flexbox and grid layout?

Flexbox : Flexbox is a one-dimensional layout model, which means it deals with arranging elements along a single axis (either horizontally or vertically). It is best suited for creating layouts in rows or columns.
Grid Layout : Grid layout is a two-dimensional layout model, allowing you to create complex layouts with both rows and columns. It divides the container into a grid of cells, making it suitable for creating multi-dimensional layouts.
